### Principles

- **Evidence first** - every claim links to data: simulations, chain analysis, on-chain metrics
- **Pro-Bitcoin** - we care about the network's health, security, and function as peer-to-peer electronic cash
- **Neutral on proposals** - we present what proposals do, how they work, and what the data shows
- **Open source** - everything is MIT licensed, no build steps, corrections welcome via PR
